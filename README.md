# Qisi Smokes

A Fabric 1.20.1 smoking / tobacco mod made for the Homestead modpack.

## Features

- Tobacco crops
- Cigarettes
- Burst capsule cigarettes
- Cigarette packs
- Cigarette butts
- Ashtray
- 2-block-tall vending machine
- Coffee / energy drinks
- Daily smoking limit
- Client-side first-person smoking animation addon

## Installation

### Client

Put both files into your Minecraft `mods` folder:

- `qisi-smokes-fabric-0.1.6.jar`
- `qisi-smokes-anim-addon-0.3.1.jar`

### Server

Only put this file into the server `mods` folder:

- `qisi-smokes-fabric-0.1.6.jar`

Do not install the animation addon on the server. It is client-only.

## Minecraft version

- Minecraft 1.20.1
- Fabric
- Built for Homestead
